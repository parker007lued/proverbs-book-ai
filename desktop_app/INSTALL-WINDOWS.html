Windows Installation Guide

Step 1: Download PowerShell Script

Right-click the file INSTALLER-WINDOWS.ps1 and select:

Run with PowerShell

Or open PowerShell and run:

powershell -ExecutionPolicy Bypass -File INSTALLER-WINDOWS.ps1

Step 2: Allow Script Execution

If you see a security warning:

  1. Click "More info"
  2. Click "Run anyway"

Or run PowerShell as Administrator and execute:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

Step 3: Wait for Installation

The installer will automatically:

  • Install Chocolatey (package manager)
  • Install Ollama (AI engine)
  • Install Pandoc (book export)
  • Install Node.js (if needed)
  • Download AI model (~2GB, takes 5-10 minutes)

Step 4: Run the App

After installation completes, run:

npm start

Or double-click the app icon if available.
