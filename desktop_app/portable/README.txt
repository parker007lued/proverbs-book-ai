# Proverbs Book AI - Portable Version

## Installation Instructions

### Option 1: Use the Mac Installer Script
1. Copy this entire folder to a Mac
2. Run: `./create-dmg.sh`
3. This creates a DMG installer

### Option 2: Manual Installation
1. Install Electron: `brew install --cask electron`
2. Double-click "Proverbs Book AI.app"
3. Or run from terminal: `open "Proverbs Book AI.app"`

### Option 3: Use Simple Installer
If you have Python, use the simple installer instead:
- Run: `python3 ../simple_installer/install.py`

## Files Included
- Proverbs Book AI.app - The application bundle
- create-dmg.sh - Script to create DMG installer (run on Mac)
- README.txt - This file

## Requirements
- macOS 10.13 or later
- Electron (installed automatically by installer script)
- Internet connection (for first-time setup)
