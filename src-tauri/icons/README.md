# Application Icons

Place your application icons in this directory. Required sizes:

- 32x32.png
- 128x128.png
- 128x128@2x.png (256x256)
- 256x256.png
- 256x256@2x.png (512x512)
- 512x512.png
- 512x512@2x.png (1024x1024)

You can generate icons from a single source image using tools like:
- [Tauri Icon Generator](https://github.com/tauri-apps/tauri-icon)
- Online icon generators
- Image editing software

For now, you can use placeholder icons or create simple ones. The app will work without custom icons, but Tauri will use default icons.
