## ğŸš€ Quick Start

### For Users

1. **Download** the app for your platform from [Download Page](website/download.html)
2. **Install** following platform-specific instructions
3. **Create Account** or login
4. **Start Writing** your Proverbs book!

### For Developers

```bash
# Clone repository
git clone https://github.com/parker007lued/proverbs-book-ai.git
cd proverbs-book-ai

# Install dependencies
cd backend-api
npm install

cd ../desktop_app
npm install

# Set up environment variables
cp backend-api/.env.example backend-api/.env
# Edit .env with your configuration

# Start backend
cd backend-api
npm start

# Start desktop app (in new terminal)
cd desktop_app
npm start
```
